1．IF NOT EXISTSを使ってpractice_review_historyテーブルを作成し、id INTEGER PRIMARY KEYとuser_id INTEGER, notes TEXT, created_at TEXTの列を定義してください。

解答
CREATE TABLE IF NOT EXISTS practice_review_history (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INTEGER,
    notes TEXT,
    created_at TEXT
);

② テーブル名決定

4．practice_courses_alterテーブルを再作成してから、description TEXT列を追加してください。
テーブルはDROPで削除後にCREATEし、その後ALTERで列を追加します。

解答
DROP TABLE IF EXISTS practice_courses_alter;
CREATE TABLE practice_courses_alter (
    id INT PRIMARY KEY,
    title TEXT,
    level TEXT
);
ALTER TABLE practice_courses_alter ADD description TEXT;

解法
① 古いpractice_courses_alterを消す
↓
② 3列で新規作成
↓
③ 後からdescription列追加


5．practice_inventory_alterテーブルをDROPで削除し、quantityとupdated_at列のみで再作成した上で、
updated_by TEXT列を追加してください。

解答
DROP TABLE IF EXISTS practice_inventory_alter;
CREATE TABLE practice_inventory_alter (
    id INT PRIMARY KEY,
    quantity int,
    updated_at date
);
ALTER TABLE practice_inventory_alter ADD updated_by TEXT;

解法
① 古いテーブルを消す
↓
② 在庫テーブル作成
↓
③ id / quantity / updated_at を定義
↓
④ 後から updated_by を追加
