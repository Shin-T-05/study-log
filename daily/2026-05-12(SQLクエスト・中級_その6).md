2．enrollmentsテーブルを使い、user_idごとの平均progressを計算し、小数第一位まで表示してください（CTE利用）。

解答
WITH progress_avg AS(
    SELECT user_id, 
    AVG(progress) AS avg_progress
    FROM enrollments
    GROUP BY user_id
)
SELECT user_id, ROUND(avg_progress, 1) AS avg_progress 
FROM progress_avg;

解法
①user_idごとにGROUP BY
↓
②AVG(progress)で平均を表示
↓
③progress_avg作成
↓
④ROUNDして表示


3．paymentsテーブルを使い、2024年の月ごとの合計支払額と決済件数をCTEでまとめてください。

解答
WITH monthly AS (
  SELECT
    substr(paid_at, 1, 7) AS month,
    amount
  FROM payments
  WHERE paid_at LIKE '2024-%'
)
SELECT
  month,
  SUM(amount) AS total_amount,
  COUNT(*) AS payment_count
FROM monthly
GROUP BY month;

解法
①payments
↓
②2024年だけ抽出
↓
③年月を切り出す
↓
④monthly作成
↓
⑤monthでGROUP BY
↓
⑥SUMとCOUNT


4．subscriptionsテーブルでstatusが"active"のユーザー割合を計算してください（CTEで分子と分母を分けて求める）。

解答
WITH total AS (
    SELECT COUNT(*) AS cnt 
    FROM subscriptions),
    active AS (
        SELECT COUNT(*) AS cnt
        FROM subscriptions 
        WHERE status = 'active')
SELECT 
ROUND(active.cnt * 100.0 / total.cnt, 1) AS active_ratio
FROM total, active;

解法
① 全契約数を数える
↓
② active契約数を数える
↓
③ 2つを並べる
↓
④ 小数1桁に丸める


5．coursesとenrollmentsを用い、カテゴリごとの受講者数を求め、受講者数が2名以上のカテゴリのみ表示してください。

解答
WITH enroll AS (
  SELECT c.category, COUNT(e.id) AS learner_count
  FROM courses c
  LEFT JOIN enrollments e
  ON c.id = e.course_id
  GROUP BY c.category
)
SELECT 
category, 
learner_count 
FROM enroll 
WHERE learner_count >= 2;

解法
① coursesを起点にする
↓
② enrollmentsを結合する
↓
③ カテゴリごとに人数を数える
↓
④ enrollという中間表を作る
↓
⑤ 2人以上だけ残す


6．feedbackテーブルでratingが3未満のコメントをTRIMして表示してください。

解答
SELECT id, TRIM(comment) AS trimmed_comment
FROM feedback
WHERE rating < 3;

解法
① feedbackテーブルを見る
↓
② ratingが3未満だけ抽出
↓
③ commentの前後の空白を削除
↓
④ 見やすく表示
