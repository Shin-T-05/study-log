1．最も注文件数が多い商品とその件数を取得してください。

解答
SELECT
    p.name,
    COUNT(o.id) AS order_count
FROM products AS p
JOIN orders AS o
ON p.id = o.product_id
GROUP BY p.id, p.name
HAVING COUNT(o.id) = (
    SELECT MAX(cnt)
    FROM (
        SELECT
            COUNT(*) AS cnt
        FROM orders
        GROUP BY product_id
    ) AS product_counts
);

解法
① 商品ごとの注文数を集計
↓
② 注文数の最大値を求める
↓
③ 最大値と一致する商品だけ残す


2．各ユーザーの平均注文金額を計算し、降順で表示してください。

解答
SELECT u.name, AVG(p.price * o.quantity) as avg_amount
FROM users AS u
JOIN orders AS o
ON u.id = o.user_id
JOIN products AS p
ON o.product_id = p.id
GROUP BY u.id,u.name
ORDER BY avg_amount DESC;

解法
① 注文ごとの金額を計算
↓
② ユーザーごとにまとめる
↓
③ 平均を取る
↓
④ 平均が高い順に並べる


3．注文金額が2番目に高い注文IDと金額を取得してください。

解答
SELECT
    o.id,
    (p.price * o.quantity) AS amount
FROM orders AS o
JOIN products AS p
ON o.product_id = p.id
WHERE (p.price * o.quantity) = (
    SELECT MAX(p2.price * o2.quantity)
    FROM orders AS o2
    JOIN products AS p2
    ON o2.product_id = p2.id
    WHERE (p2.price * o2.quantity) < (
        SELECT MAX(p3.price * o3.quantity)
        FROM orders AS o3
        JOIN products AS p3
        ON o3.product_id = p3.id
    )
);

解法
① 最大注文金額を求める
↓
② それ未満に絞る
↓
③ その中で最大を求める
↓
④ 一致する注文を取得
