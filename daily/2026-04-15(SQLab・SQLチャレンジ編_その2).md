7．出版タイトル数が多い著者TOP3を取得してください。また、著者名は昇順で並び替えてください。出力項目はname(著者名)とpublished_title_num(出版タイトル数)です。

解答
SELECT a.name,COUNT(*)AS published_title_num
FROM authors AS a
JOIN book_authors AS ba
ON a.id = ba.author_id
JOIN books AS b
ON b.id = ba.book_id
GROUP BY a.name
ORDER BY published_title_num DESC,a.name
LIMIT 3;

解法
① 著者と本の関係を結合
↓
② 著者ごとにまとめる
↓
③ 出版数をカウント
↓
④ 多い順に並べる
↓
⑤ 上位3人を表示


8．複数のカテゴリーに属している書籍名の一覧を取得してください。出力項目はname(書籍名)です。

解答
SELECT b.name
FROM books AS b
JOIN book_categories AS bc
ON b.id = bc.book_id
WHERE bc.book_id IN(SELECT book_id
                   FROM book_categories
                   GROUP BY book_id
                   HAVING COUNT(*) >1)
GROUP BY b.name;

解法
① book_categoriesから「カテゴリ2つ以上の本」を抽出
↓
② booksと結合
↓
③ 該当本だけ残す
↓
④ 重複をまとめる
↓
⑤ 本の名前を表示


9．書籍名に「宇宙」または「星」を含んでおり、かつ著者が女性の書籍名を取得してください。出力項目はname(書籍名)です。

解答
SELECT b.name
FROM authors AS a
JOIN book_authors AS ba
ON a.id = ba.author_id
JOIN books AS b
ON b.id = ba.book_id
WHERE (b.name LIKE '%宇宙%' OR b.name LIKE '%星%')
AND a.gender = '女性';

解法
① 著者と本をJOIN
↓
② タイトルに「宇宙 or 星」を含む本を抽出
↓
③ 女性著者に限定
↓
④ 本の名前を表示


10．idが1のイベントを削除し、idが2のイベントの最大人数を200に更新してください。
イベントテーブルに「イベント名：古本まつり、最大人数：75人」のイベントを追加してください。
イベント一覧を取得してください。

解答
DELETE FROM events
WHERE id = 1;

UPDATE events 
SET max_num = 200
WHERE id = 2;

INSERT INTO events VALUES(3,'古本まつり',75);

SELECT *
FROM events;

解法
① id=1を削除
↓
② 残った全データのmax_numを200に更新
↓
③ 新しいイベントを追加
↓
④ 最終状態を表示
