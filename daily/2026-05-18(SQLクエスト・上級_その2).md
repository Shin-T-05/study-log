1．累計購入金額が2000円以上のユーザー全員に、商品「ペン」を1本（注文日2024-02-10）practice_insert_ordersテーブルへ追加してください。
GROUP BYとHAVINGを用いたサブクエリを活用します。

解答
INSERT INTO practice_insert_orders (user_id, product_id, quantity, order_date)
SELECT
    u.id,
    (
        SELECT id 
        FROM products 
        WHERE name = 'ペン'
    ),
    1,
    '2024-02-10'
FROM users AS u
WHERE EXISTS (
    SELECT 1
    FROM orders AS o
    JOIN products AS p
    ON o.product_id = p.id
    WHERE o.user_id = u.id
    GROUP BY o.user_id
    HAVING SUM(o.quantity * p.price) >= 2000
);

解法
① 全ユーザーを見る
↓
② 各ユーザーの累計購入額を計算
↓
③ 2000円以上の人だけ残す
↓
④ ペンの商品IDを取得
↓
⑤ 注文データを作る
↓
⑥ INSERTする


2．全商品の平均価格より高い商品を取得してください。

解答
SELECT *
FROM products
WHERE price > (
    SELECT AVG(price)
    FROM products
);

解法
① 比較基準を作る（平均価格）
↓
② 各商品の価格と比較する
↓
③ 平均より高いものだけ残す


3．電子機器カテゴリの商品を少なくとも1度注文したユーザーを取得してください。

解答
SELECT DISTINCT u.name
FROM users AS u
WHERE u.id IN (
    SELECT o.user_id
    FROM orders AS o
    JOIN products AS p
    ON o.product_id = p.id
    WHERE p.category = '電子機器'
    GROUP BY o.user_id
    HAVING COUNT(*) >= 1
);

解法
① 電子機器の注文履歴を探す
↓
② ユーザー単位にまとめる（1回以上）
↓
③ 対象user_id一覧を作る
↓
④ usersから名前を取る
↓
⑤ 同名があれば重複排除
