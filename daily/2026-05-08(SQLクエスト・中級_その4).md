1．practice_update_ordersテーブルでinventoryテーブルのquantityが0のproduct_idに対するstatusを"pending_stock"に更新してください。

解答
UPDATE practice_update_orders
SET status = 'pending_stock'
WHERE product_id IN (
    SELECT product_id FROM inventory WHERE quantity = 0
);

解法
① inventoryを見る
↓
② 在庫0の商品一覧を作る
↓
③ ordersで一致する商品を探す
↓
④ status更新


2．enrollmentsテーブルでcoursesテーブルのlevelが"beginner"の講座に登録している受講者のprogressを0にリセットしてください。

解答
UPDATE enrollments
SET progress = 0
WHERE course_id IN (
    SELECT id FROM courses WHERE 
    level = 'beginner'
);

解法
① coursesを見る
↓
② beginnerコース一覧を作る
↓
③ enrollmentsで一致するcourse_idを探す
↓
④ progressを0に更新


3．subscriptionsテーブルでstarted_atが2023-01-01以前のユーザーに対してpractice_update_usersテーブルのtagを"loyal"に更新してください。

解答
UPDATE practice_update_users
SET tag = 'loyal'
WHERE id IN (
    SELECT user_id FROM subscriptions 
    WHERE started_at <= '2023-01-01'
);

解法
① subscriptionsを見る
↓
② 長期契約user_id一覧を作る
↓
③ usersで一致する人を探す
↓
④ loyalタグ付け


4．practice_delete_notificationsテーブルでユーザーがpractice_update_usersテーブルでstatusが"deleted"になっている場合、その通知を削除してください。

解答
DELETE FROM practice_delete_notifications 
WHERE user_id IN (
    SELECT id FROM practice_update_users 
    WHERE status = 'deleted'
);

解法
① deletedユーザー一覧を作る
↓
② 通知テーブルで一致するuser_idを探す
↓
③ 該当通知を削除
